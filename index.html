Setup Guide

Connect Google Sheets as your free database

Follow these 5 steps once. After setup every user can log in from any phone or computer and data syncs automatically through your Google Sheet.

1

Create a Google Sheet

Go to sheets.new — a new spreadsheet opens. Name it Budget Hub.

At the bottom, add two tabs: right-click the default tab → Rename to Users, then click the + button and name the new tab Entries.

2

Add headers to both sheets

Click the Users tab. In row 1, type these in separate columns A to I:

username  pin  role  salary  payday  rent  tuition  gym  efund

Then in row 2 add your first admin user (one value per column):

Khasnen  Khasnen123  admin  162000  10  18000  0  0  5000

Click the Entries tab. In row 1 type these in columns A to G:

id  user  type  category  amount  note  date

3

Create the Apps Script (connects app ↔ Sheet)

In your Google Sheet, click Extensions → Apps Script. A new tab opens.

Delete everything in the editor and paste this entire script:

Click the Save icon (💾), then click Deploy → New deployment.

In the dialog: Type = Web app · Execute as = Me · Who has access = Anyone · Click Deploy.

Google will ask you to authorize — click through. Then copy the Web app URL that appears.

4

Paste your Web App URL here

Paste the URL you copied from step 3 below, then click Save & Test:

5

Host on GitHub Pages (free, share with everyone)

First download this app file:

Then go to github.com → Sign up free → New repository → name it budget-hub → set to Public.

Click uploading an existing file → upload the index.html you downloaded → Commit.

Go to Settings → Pages → Source: Deploy from branch → main → / (root) → Save.

Wait 1-2 minutes. Your app is live at: https://YOUR-USERNAME.github.io/budget-hub

Share that link with Sanan, Sahad and everyone else. They open it on their phone and log in.
